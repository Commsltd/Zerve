# Zerve Project

This project contains data validation utilities for the Zerve financial model.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Run the main script to load data and execute validations:

```bash
python main.py
```

## Project Structure

- `src/` - Source code directory
  - `validations/` - Validation modules
  - `data_loader.py` - Data loading utilities
  - `data_models.py` - Data model definitions
- `tests/` - Test suite
- `data/` - Sample data files
- `main.py` - Main execution script
- `requirements.txt` - Python dependencies
